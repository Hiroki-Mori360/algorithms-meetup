# Cellphone Texting

## Summary

Mobile phones have letters associated with their number keypads like so:
{ 1: '', 2: 'ABC', 3: 'DEF',
4: 'GHI', 5: 'JKL', 6: 'MNO',
7: 'PQRS', 8: 'TUV', 9: 'WXYZ',
0: ' '}

Given a text string, determine which numbers must be pressed in order to produce the desired text

# Inputs
'hello world'

# Outputs
4433555555666096667775553
